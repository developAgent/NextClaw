# ClawX 100% 功能对照清单

## 项目概述

**ClawX** 是基于 [OpenClaw](https://github.com/OpenClaw) 的图形化桌面客户端，将命令行式的 AI 编排转变为易用、美观的桌面体验。

**技术栈**：
- Electron 40+（桌面框架）
- React 19 + TypeScript（前端）
- OpenClaw 核心（AI 运行时）
- Tailwind CSS + shadcn/ui（UI）
- Zustand（状态管理）
- Vite（构建工具）

## 与原项目的主要差异

### 当前 CEOClaw vs ClawX

| 特性 | CEOClaw（当前） | ClawX（目标） |
|------|----------------|---------------|
| 框架 | Tauri 2.0 | Electron 40+ |
| AI 核心 | 无 | OpenClaw |
| 用途 | 桌面自动化 | AI 助手图形界面 |
| 频道 | 无 | 8+ 集成 |
| 技能系统 | 无 | 完整实现 |
| 定时任务 | 无 | Cron 支持 |

### 架构差异

**ClawX**:
```
Electron Main Process
    ↓
OpenClaw Gateway (127.0.0.1:18789)
    ↓
React Renderer Process
```

**CEOClaw（当前）**:
```
Tauri Backend (Rust)
    ↓
React Frontend
```
